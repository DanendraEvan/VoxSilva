Untuk membuat aplikasi PWA yang dapat diinstall, Anda perlu membuat ikon dengan ukuran berikut:

  1. icon-192x192.png - Ukuran 192x192 pixels (untuk Android)
  2. icon-512x512.png - Ukuran 512x512 pixels (untuk Android & Splash Screen)
‚ö†Ô∏è Penting: File ikon harus ditempatkan di folder public/ agar dapat diakses oleh PWA.

üîß Metode 1: Menggunakan Online Tool (Paling Mudah)

  1. Buka RealFaviconGenerator atau PWA Builder Image Generator
  2. Upload logo VoxSilva Anda (logo.png dari folder src/components/Header/)
  3. Pilih ukuran 192x192 dan 512x512
  4. Download ikon yang dihasilkan
  5. Rename file menjadi icon-192x192.png dan icon-512x512.png
  6. Simpan kedua file di folder public/

üîß Metode 2: Menggunakan Image Editor (Photoshop/GIMP)

  1. Buka logo VoxSilva di editor gambar
  2. Resize ke 192x192 pixels, simpan sebagai icon-192x192.png
  3. Resize ke 512x512 pixels, simpan sebagai icon-512x512.png
  4. Pastikan format PNG dengan transparansi (jika diperlukan)
  5. Simpan kedua file di folder public/

üîß Metode 3: Menggunakan Command Line (ImageMagick)

Jika Anda memiliki ImageMagick terinstall:

# Convert logo ke icon 192x192 convert src/components/Header/logo.png -resize 192x192 public/icon-192x192.png # Convert logo ke icon 512x512 convert src/components/Header/logo.png -resize 512x512 public/icon-512x512.png

üîß Metode 4: Menggunakan Node.js Script

Install sharp terlebih dahulu:

npm install sharp --save-dev

Kemudian buat file scripts/generate-icons.js:

Jalankan dengan:

node scripts/generate-icons.js
